Film liste app.

En progressiv web-applikasjon for å holde oversikt over filmer du har sett eller planlegger å se.

Funksjonalitet
Filmoversikt: Se en liste over dine lagrede filmer.
Vurdering: Gi filmer terningkast fra 1 til 10.
Kommentarer: Legg til personlige notater om hver film.
Offline-støtte: Appen fungerer uten nett takket være lokal lagring.
Deling: Mulighet for å dele listen via direkte link eller e-post.
